# EDGE logo assets

High-resolution logo files for the SM EDGE brand. Use these for:

- **Social / OG images** — `edge-full.png` (with “SPORTS MOCKERY” tagline)
- **Nav / headers** — Prefer the scalable `EdgeLogo` component in `@/components/edge/EdgeLogo.tsx`, or `edge-compact.png` / `edge-sm.png` for pixel-perfect match
- **EDGE+ / premium** — `edge-plus.png`
- **Minimal wordmark** — `edge-dash.png` (EDGE only)

| File | Description |
|------|-------------|
| `edge-plus.png` | EDGE+ wordmark (blue gradient + gold +) |
| `edge-dash.png` | EDGE only, compact |
| `edge-compact.png` | SM + star + EDGE |
| `edge-full.png` | SM + star + EDGE + “SPORTS MOCKERY” tagline |
| `edge-sm.png` | SM + star + EDGE (alternate) |
| `edge-sm-2.png` | SM + star + EDGE (alternate 2) |

For in-app use at any size, the SVG in `EdgeLogo` is resolution-independent.
